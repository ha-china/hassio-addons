# Home Assistant Add-on: Pocket ID

基于 [Pocket ID](https://pocket-id.org/) 的 OIDC 的 Home Assistant 插件，用于通过 OIDC 进行密钥认证。

## 关于

Pocket ID 是一个简单的 **OIDC 提供商**，用于 **密钥认证**。此插件允许您集成使用 WebAuthn/FIDO2 的无密码登录，提供无缝且安全的认证体验。
