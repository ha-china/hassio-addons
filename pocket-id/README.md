# Home Assistant Add-on: Pocket ID

Home Assistant 插件，基于 [Pocket ID](https://pocket-id.org/)，实现基于 OIDC 的密钥认证。

## 关于

Pocket ID 是一个简单的 **OIDC 提供者**，用于 **密钥认证**。该插件允许您使用 WebAuthn/FIDO2 集成无密码登录，提供无缝且安全的认证体验。
