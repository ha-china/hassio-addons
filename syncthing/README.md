# Syncthing 扩展

[**Syncthing**](https://syncthing.net/) 被包裹在 [Home Assistant Supervisor 扩展](https://www.home-assistant.io/addons/) 内。

> Syncthing 是一个持续文件同步程序。它实时地在两台或多台计算机之间同步文件，并安全地保护免受窥视。你的数据属于你，你值得选择数据的存储位置，是否与第三方共享，以及如何通过互联网传输数据。
