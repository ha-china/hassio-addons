# Home Assistant Add-on: Authelia

Web应用的单一登录多因素门户

## 关于

Authelia是一个开源的身份验证和授权服务器，通过一个Web门户为您的应用程序提供双因素认证和单一登录（SSO）。它作为反向代理的伴侣，允许、拒绝或重定向请求。
