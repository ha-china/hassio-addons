# 音乐助手（测试版）插件

音乐助手的官方测试版发布渠道。

## ⚠️ 重要通知

这是一个音乐助手的**测试版**。它包含在稳定版发布前进行测试的新功能和改进。

**如果您：**

- 希望提前访问新功能
- 愿意帮助测试和报告问题
- 可以忍受偶尔的bug或不稳定性
- 希望为使音乐助手变得更好做出贡献

**请不要使用此插件如果您：**

- 需要一个始终稳定的、生产就绪的系统
- 不舒服于解决问题
- 无法承受音乐设置中的任何停机时间

## 什么是测试版？

测试版是功能完整的版本，在成为稳定版之前进行测试。它们通常包括：

- ✨ 稳定版中尚未包含的新功能
- 🔧 性能改进
- 🐛 之前版本的bug修复
- 🧪 需要实际测试的更改

## 与稳定版的差异

| 方面    | 稳定版               | 测试版                                  |
|---------|--------------------|-------------------------------------|
| 稳定性  | 非常稳定        | 通常稳定但有可能会出现问题 |
| 功能  | 经过充分测试的功能 | 正在测试的新功能             |
| 更新   | 更少频繁        | 更频繁                         |
| 用途  | 生产           | 测试和早期采用              |

## 报告问题

作为一名测试版测试者，您的反馈非常有价值！请报告您遇到的问题：

### 报告前

1. 检查插件日志（如果需要，可以全局启用或按提供者级别启用`debug`日志）
2. 搜索[现有问题](https://github.com/music-assistant/support)
3. 如果可能，验证问题在稳定版中是否发生

### 报告时

包括：

- 📋 重现问题的步骤
- 📝 插件的完整日志（或从MA的Web界面下载完整日志文件）
- 🔢 音乐助手版本（在Web UI中可见）
- 🎵 您正在使用的音乐提供者
- 🔊 受影响的播放器

**报告地点**：[GitHub支持仓库](https://github.com/music-assistant/support)

## 更新

测试版比稳定版更新更频繁。通常每周更新一到多次。

## 已知的限制和注意

- 测试版可能会有破坏性更改
- 某些功能可能只有部分实现
- 版本之间可能会发生数据库迁移
- 性能优化可能仍在进行中
- 您不能从稳定版迁移（反之亦然）

提示：如果您想测试测试版，同时保留稳定版，只需停止稳定版插件并运行测试版插件。要恢复到稳定版，只需再次停止测试版插件并启动稳定版。两个插件不能同时激活。

## 测试版最佳实践

1. **定期备份**：始终维护最近的备份
2. **监控日志**：留意日志中的问题
3. **报告问题**：通过报告错误帮助我们改进
4. **保持耐心**：某些功能可能无法完美工作
5. **保持更新**：安装更新以获取最新的修复

在Home Assistant中备份音乐助手插件也会包括您的音乐助手数据。请确保在更新到新版本前始终进行备份，以便您可以轻松地恢复到以前的版本！

## 回滚策略

### 如果事情出错了

1. **停止插件**
2. **从备份恢复**（您应该已经备份了，对吧？）
3. **报告问题**
