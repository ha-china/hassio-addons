## 关于

TasmoAdmin（曾被称为 SonWEB）是一个管理所有您通过 Sonoff-Tasmota 烧录的设备的管理式网络界面。
它的部分功能：

- 扫描您的网络并自动添加您的设备
- 快速轻松地查看所有您设备的状态
- 从一个地方配置所有您的设备
- 同时通过无线方式向一个或多个您的设备发送固件更新
- 可以自动为您下载最新的固件
