# Home Assistant 拓展程序：taptap - Tigo CCA 到 MQTT

Tigo CCA Tap 到 MQTT 的实现，用于在本地从 Tigo 优化器获取数据到 Home Assistant。有关所需的硬件安装和使用说明，请参阅 [文档](https://github.com/litinoveweedle/hassio-addons/blob/main/taptap/DOCS.md)。
