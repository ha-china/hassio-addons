## 关于

AdGuard Home 是一个具有家长控制（成人内容阻止）功能的网络范围广告和跟踪器阻止DNS服务器。它的目的是让您控制整个网络和所有您的设备，并且不需要使用客户端程序。

AdGuard Home 提供了一个美观、易于使用且功能丰富的网页界面，以便轻松管理过滤过程及其设置。
